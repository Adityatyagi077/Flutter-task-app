# 📱 My Tasks – Flutter Task Manager App

A beautifully designed task management application built using Flutter. This app helps users organize daily tasks, track progress, and stay productive with a clean and modern user interface.

---

## ✨ Features

* ✅ Add, update, and manage daily tasks
* 📊 Track daily progress (completion percentage)
* 🕒 Task scheduling with time and date
* 🎨 Clean and modern UI with smooth design
* 🔔 Notification icon (for future enhancements)
* 📱 Responsive layout for different screen sizes

---

## 🖼️ UI Preview

* Gradient header with greeting and progress tracker
* Task cards with color indicators
* Floating "Add Task" button
* Bottom navigation bar for easy access

---

## 🛠️ Built With

* 💙 Flutter
* 🎯 Dart
* 📦 Material UI Components
